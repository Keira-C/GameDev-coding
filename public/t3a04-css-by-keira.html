List style inside

  • This list style puts the bullet points inside the list items using the property list-style-position: inside. It will push the text at the start because it is part of the list item. The default bullet point position is usually outside the list item. This means that the start of each line of a list item are in line vertically. There are many cool things that you can do with lists.
